# Web Development Internship - Technical Assessment (IF Solutions)

This repository contains my submission for the Web Development Internship practical assignment. The task was to convert the provided visual guide into a clean, fully responsive, and pixel-perfect homepage layout.

Live Preview: GitHub repo Link : https://github.com/anu2256/IF-Solutions-Task.git

*Tech Stack & Libraries Used*

- HTML5: Structured using modern Semantic HTML semantic tags (<section>, <header>, <footer>, etc.) for improved SEO and accessibility.
- Tailwind CSS: Utilized via modern utility classes for rapid, responsive styling and customized modern components.
- Lucide Icons: Used for clean, crisp, and high-quality vector icons across the layout.

Key Features Implemented
- *Fully Responsive Design:*  Fully optimized for Mobile, Tablet, and Desktop screens using Tailwind's breakpoint system.
- *Interactive Micro-interactions:*  Added smooth CSS transitions and hover effects on product cards, call-to-action buttons, and thumbnail images.
- *Creative Enhancements:* Integrated unique UI design practices (like the custom smooth heart-shaped image mask and advanced drop-shadow depths) to elevate the visual appeal.
- *Clean Codebase:* Written manually from scratch following standard web practices, keeping the code human-readable and logical.
